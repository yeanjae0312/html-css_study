html태그학습_07이미지태그

이미지태그는 웹문서에 이미지를 표시하기 위하여 사용하는 태그이며 source의 약자인 'src'속성과 이미지가 나오지 않을 때 대체할 문자인 'alt'속성을 사용한다.

웹에서 사용하는 이미지의 종류에는 jpg(고화질사진), png(w3c에서 만든 이미지 포멧, 고화질, 투명배경이미지, 로고, 아이콘), gif(단순한 색상이미지, 애니메이션, 아이콘, 로고)가 있으며, html5에서 추가된 svg이미지도 사용된다.

이미지태그는 다른태그와 달리 닫는태그가 없으며,
img src="이미지 경로나 주소"로 작성된다.

1. 이미지 삽입하기

절대주소를 사용하여 이미지 삽입


상대주소를 사용하여 이미지 삽입

URL주소 - 웹서버에 위치한 파일이나 문서가 위치한 정확한 경로나 주소, 위치를 말함.

주소는 절대주소와 상대주소가 있으며 절대주소는 http://~로 시작하는 주소, 상대주소는 현재문서의 위치를 기준으로 주소가 결정됨.
./폴더명/파일명.확장자 => 문서를 기준으로 하위폴더일 경우
../폴더명/파일명.확장자 => 문서를 기준으로 폴더가 다를 때(상위폴더)
파일명.확장자 => 문서를 기준으로 같은 위치에 있을 때

url작성법 : http://도메인명/폴더명/파일명.확장자

문서와 이미지가 같은 위치에 있을 경우


문서와 이미지가 각각 다른 폴더에 위치한 경우


스타벅스 로고를 0617폴더에 img안에 logo폴더를 만들고 저장하여 삽입하시오.

2. 이미지 주소(경로)가 잘못된 경우

alternate text의 약자로서 이미지가 나오지 않을 때 사용자가 이해할 수 있는 대체문자를 말하며, 시각장애인이 스크린리더를 통해서 음성으로 이해할 수 있는 문자임. 웹접근성을 위하여 반드시 삽입해야한다. W3C HTML마크업 검사시 오류가 나오기 때문에 반드시 사용해야 함.

3. 이미지 크기지정 속성 - width(가로), height(세로)


크기속성을 넣어주면 브라우저가 따로 이미지의 크기를 계산하지 않고 보여주지만, 크기값이 없으면 계산을 해야하기 때문에 페이지 로딩속도에 영향을 줄 수 있다.

4. 반응형 이미지 만들기

반응형 이미지는 브라우저의 가로폭을 줄이면 이미지도 같이 작아지며, 늘리면 반대로 이미지 크기도 커지는 기술. 가로세로가 비율유지되면서 자동으로 조정이 된다.

5. 이미지를 대신하여 표시해주는 사이트 - placeholder.com

웹 기획시 레이아웃 작성할 때 이미지를 제작하지 않고 이미지처럼 가로세로 크기로 표시해주는 사이트이다.


6. html5에서 추가된 이미지 태그

이미지에 설명글을 넣기 위한 태그요소가 새로 추가되었으며, figure, figcaption태그로 작성한다.
